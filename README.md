# Monte Carlo Simulation for Portfolio Risk and Stock Price Uncertainty

## Overview

This project explores the use of Monte Carlo simulation to model uncertainty in financial markets.

The project is divided into two parts:

**Part 1** investigates portfolio risk by estimating the probability that a portfolio loses money over a one-year investment horizon.

**Part 2** evaluates whether Monte Carlo simulations based on historical stock returns can generate realistic prediction intervals for future stock prices.

Together, these analyses demonstrate how simulation can be used to quantify uncertainty, assess risk, and evaluate statistical models using real-world financial data.

---

## Project Motivation

This project was developed to learn and apply statistical concepts through a practical real-world problem. The goal was to explore Monte Carlo methods, statistical modeling, uncertainty quantification, and model evaluation using real financial data.

---

## Part 1: Portfolio Risk Analysis

### Research Question

> What is the probability that a portfolio loses money over a one-year period?

### Methodology

A portfolio consisting of:

* 50% Apple
* 50% Coca-Cola

was constructed using historical price data.

Daily returns were used to estimate:

* Mean daily return
* Daily volatility

Using these estimates, 10,000 Monte Carlo simulations were generated over a one-year horizon (252 trading days).

The simulated outcomes were then used to estimate:

* Probability of loss
* Expected portfolio value
* Distribution of possible outcomes

### Key Result

The simulation estimated the probability that the portfolio would finish the year below its initial value and illustrated the range of possible future portfolio values implied by the model.

## Part 2: Evaluating Prediction Intervals

### Research Question

> Can Monte Carlo simulations based on historical returns generate realistic prediction intervals for future stock prices?

### Methodology

Historical data from April 2024 to April 2025 was collected for:

* Apple (AAPL)
* Microsoft (MSFT)
* Coca-Cola (KO)
* JPMorgan Chase (JPM)
* Exxon Mobil (XOM)
* Walmart (WMT)

For each stock:

1. Daily returns were calculated.
2. Mean return and volatility were estimated.
3. 10,000 Monte Carlo simulations were generated for the following year.
4. Prediction intervals were constructed from the simulated distributions.
5. Realized stock prices from April 2026 were compared against the simulated outcomes.

### Model Evaluation

Two complementary evaluation methods were used:

#### Coverage Analysis

Determines whether the realized stock price falls within the prediction interval generated by the simulation.

#### Percentile Analysis

Determines where the realized stock price falls within the simulated distribution.

This provides a more detailed assessment of model calibration than interval coverage alone.

## Model Assumptions

The simulation relies on several simplifying assumptions:

* Future returns behave similarly to historical returns.
* Daily returns follow a normal distribution.
* Historical mean returns and volatility remain stable over the prediction period.
* Market shocks and structural changes are not explicitly modeled.

These assumptions make the model simple and interpretable, while also highlighting some of the limitations of simulation-based forecasting.
